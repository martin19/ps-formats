# photoshop file formats

Public documentation is availabale at:
https://www.adobe.com/devnet-apps/photoshop/fileformatashtml/

Readers for these formats are part of this package:
- brush (.abr)
- swatch (.ase)
- layer style (.asl)
- shape (.csh)
- gradient (.grd)
- pattern (.pat)
- contour (.shc)
- document (.psd)

Install dependencies
```shell
npm install
```

Build the project
```shell
npm run build
```

Run tests
```shell
npm test
```
